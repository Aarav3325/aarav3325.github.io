Projects

Selected work

01
GeoWav

Developed a location-based Android application that detects user entry and exit from selected places using reliable foreground and background location tracking. Integrated Google Places API for selecting locations and the Fused Location Provider for accurate real-time updates, along with Snap-to-Road path rendering to display smooth and realistic travel routes on maps. Implemented real-time live location sharing and an emergency mode using Firebase Realtime Database, enabling secure multi-user synchronization and instant updates between participants. Built a timeline and session history system that records trips, stay points, and location events with timestamps and stores them securely in the realtime database, along with a playback feature that allows users to visually replay their recorded routes and movements on the map. Developed a foreground notification service to maintain continuous location tracking, live location sharing, and route updates while ensuring reliable background execution. Applied MVVM architecture with Hilt dependency injection to manage services, permissions, location state handling, and scalable background task management.
